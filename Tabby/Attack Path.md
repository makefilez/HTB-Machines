## Upgrading to a SSH Session
Lets generate the ssh keys to make the session an ssh sssion

```shell
ash@tabby:~$ mkdir .ssh

ash@tabby:~$ cd .ssh

Generating public/private rsa key pair.

ash@tabby:~/.ssh$ cat id_rsa

ash@tabby:~/.ssh$ cat id_rsa.pub > authorized_keys

## Attack BOX ##

ssh -i id_rsa ash@10.10.10.194
```
